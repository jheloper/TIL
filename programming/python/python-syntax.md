# 파이썬 조건문과 반복문

# 조건문

- 다음과 같은 형태로 사용한다. switch case문은 지원하지 않는다.
```python
if condition1 and condition2:
    event1.start()
elif condition3 or condition4:
    event2.start()
else:
    event3.start()
```

# 반복문
- while 문.
```python
i = 0
while i < 10:
    print("I Like Python!")
    i = i + 1
```
- for 문.
```python
number_list = [1, 2, 3, 4, 5]
for number in number_list:
    print(number)
```
